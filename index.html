Instellingen

Voor échte resultaten op deze pagina gebruikt Oisearch de gratis Google Custom Search JSON API (100 zoekopdrachten/dag). Maak een Programmable Search Engine aan (kies "zoek op het hele web"), pak je Search engine ID (cx), en genereer een API-sleutel. Zonder sleutel verwijst Oisearch door naar je gekozen fallback-zoekmachine.
Gebruikt wanneer er geen API-sleutel is ingesteld.
Interface-taal (hl) en regio (gl) voor resultaten.
